Villkor Telenor Mobilstart Nedanstående villkor måste godkännas innan några tjänster kan startas Mobilstart ägs av Telenor Sverige AB, org nr 556421-0309 Villkor för nyttjande av tjänster på Mobilstart 1. För att registrera ditt företag och bli kund hos Mobilstart krävs ett giltigt F-skattebevis. Ideella föreningar och liknande skall presentera intyg från Skattemyndigheten om skattebefrielse. 2. Som användare av Mobilstart förbinder du dig att läsa och följa de regler som är uppställda av Etiska Rådet och bransch­organsationen Morgan. Vid start av eventuella tävlingar skall också Lotteriinspektionens regler följas. 3. Kunden förbinder sig att inte starta tjänster med innehåll som inte ligger inom ramen för vad som kan anses vara god etik och moral. Exempel på icke tillåtet innehåll: * a.Sexuellt avvikande handlingar * b.Främlingsfientligt material * c.Våldshandlingar 4. Kunden ansvarar själv för de tjänster som lanserats via Mobilstart. I ansvaret ligger bland annat: * a.tillse att innehållet som lanseras tillhör kunden, eller att erforderliga tillstånd/avtal inhämtats för vidareförsäljning av 3:e parts innehåll * b.tillse att eventuella royalties eller avgifter till exempelvis STIM betalas * c.tillse marknadsföring av tjänsten endast sker i medier som tillhör kunden, eller där erforderliga tillstånd/avtal inhämtats. 5. Kunden får ej använda Mobilstart för att skicka s.k. SPAM. Meddelanden som kan uppfattas som hotfulla eller stötande av mottagaren får heller inte distribueras via Mobilstart. 6. Kunden ansvarar själv för slutkundssupport av de tjänster som lanserats via Mobilstart. Det innebär att Kundens slutkund skall kunna kontakta Kunden avseende frågor om till exempel tjänstens innehåll, kvalitet och pris via de kontaktuppgifter som används vid registrering av konto på Mobilstart. 7. Mobilstart har rätt att skicka information, som är relaterad till Mobilstarts produkter, via de uppgifter som kunden lämnat vid registreringen. Kunden äger således inte rätt föra in reklambudskap för andra produkter än Kundens egna genom Mobilstart. 8. Mobilstart har rätt att radera konton som har varit inaktiva i mer än 6 månader. Anledningen till detta är att frigöra prefix som inte används. Kunden meddelas alltid innan en eventuell radering. 9. Om utbetald ersättning visar sig vara tillskansad genom någon form av bedrägeri äger Mobilstart rätten att kräva återbetalning av pengarna från kunden. 10. Mobilstart äger rätten att ändra villkoren för användandet av Mobilstart om detta krävs för att efterleva lagar, regler, samt myndighetsbeslut. Om ändringarna är av betydande karaktär kommer detta meddelas kunden. Om kunden fortsätter att använda Mobilstart efter genomförda ändringar kommer detta att tolkas som att kunden har godkänt ändringarna. Kunden kan när som hels avsluta sitt engagemang med Mobilstart. 11. Mobilstart är begränsat till ett prefix per organisationsnummer och delat kortnummer. 12. Telenor förbehåller sig rätten att radera konton som sätts upp med prefix kopplade till redan existerande varumärken där kontoägaren ej representerar detta varumärke. 13. I de fall då ovanstående villkor ej efterlevs äger Telenor Sverige AB rätt att temporärt eller permanent stänga av Kunden från vidare användning av Mobilstart samt innehålla eventuella ej utlösta intäkter.
